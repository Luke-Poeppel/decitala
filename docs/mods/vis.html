vis¶

decitala.vis.create_tree_diagram(FragmentTree, path=None, pdf_path=None, verbose=False)[source]¶

This function creates a visualization of a given FragmentTree using the Treant.js library. If a path is provided, all the files will be stored there. Otherwise, they will be stored in a tempfile for the display.

Parameters

  • FragmentTree (FragmentTree) – A Fragment tree

  • path (str) – path to the folder where the visualization will be stored.

Returns

folder at the provided path containing an index.html file which has a visualization of the provided FragmentTree.

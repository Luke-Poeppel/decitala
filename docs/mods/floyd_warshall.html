floyd_warshall¶

Implementation of the Floyd-Warshall Algorithm (path of minimal cost).

decitala.path_finding.floyd_warshall.cost(vertex_1, vertex_2, weights)[source]¶

Cost function used in the Floyd-Warshall Algorithm.

Parameters

  • vertex_1 (GeneralFragment) – an object inheriting from GeneralFragment.

  • vertex_2 (GeneralFragment) – an object inheriting from GeneralFragment.

  • weights (dict) – weights used in the model. Must sum to 1. Requires “gap” and “onsets” values.

Returns

cost of moving from vertex_1 to vertex_2.

Return type

float

decitala.path_finding.floyd_warshall.floyd_warshall(data, weights)[source]¶

Calculates the distance and next matrices of the Floyd-Warshall path-finding algorithm.

Parameters

  • data (list) – data from rolling_search.

  • weights (dict) – weights to be used in the cost function. Must sum to 1. Requires “gap” and “onsets” values.

Returns

two matrices of size len(data) x len(data): first is the weighted adjacency matrix, the second is the matrix used for path reconstruction.

Return type

tuple

class decitala.fragment.GeneralFragment(data, name=None, **kwargs)[source]¶

Bases: object

Class representing a generic rhythmic fragment. The user must provide either a path to a music21 readable file or an array of quarter length values.

Parameters

  • data – input data to GeneralFragment (either an array of quarter length values or a path to a music21 readable file.)

  • name (str) – optional name argument.

Raises

FragmentException – when an array and file are provided or neither are provided.

>>> random_fragment_path = '/Users/lukepoeppel/decitala/Fragments/Decitalas/63_Nandi.xml'
>>> g1 = GeneralFragment(data=random_fragment_path, name='test')
>>> g1
<fragment.GeneralFragment test: [0.5  0.25 0.25 0.5  0.5  1.   1.  ]>
>>> g1.filename
'63_Nandi.xml'
>>> g1.coolness_level = 'pretty cool'
>>> g1.coolness_level
'pretty cool'

morris_symmetry_class()[source]¶
Returns

the fragment’s form of rhythmic symmetry, as defined by Morris in Sets, Scales and Rhythmic Cycles. A Classification of Talas in Indian Music (1999).

Return type

str

    1. Maximally Trivial: of the form \(X\) (one onset, one anga class)

    1. Trivial Symmetry: of the form \(XXXXXX\) (multiple onsets, same anga class)

    1. Trivial Dual Symmetry: of the form \(XY\) (two onsets, two anga classes)

    1. Maximally Trivial Palindrome: of the form \(XXX...XYX...XXX\) (multiple onsets, two anga classes)

    1. Trivial Dual Palindromic: of the form \(XXX...XYYYX...XXX\) (multiple onsets, two anga classes)

    1. Palindromic: of the form \(XY...Z...YX\) (multiple onsets, \(n/2\) anga classes)

    1. Stream: of the form \(XYZ...abc...\) (\(n\) onsets, \(n\) anga classes)

classmethod get_by_id(input_id)[source]¶

A class method which retrieves a Decitala object based on a given ID number. These numbers are listed in the Lavignac Encyclopédie (1921) and Messiaen Traité. Some talas have (as I’m calling them) “sub-talas,” meaning that their id is not unique. Querying by those talas is currently not supported.

Returns

a Decitala object

Parameters

input_id (int) – id number of the tala (in range 1-120).

Return type

Decitala

Raises

DecitalaException – when there is an issue with the input_id.
